# Advanced Quote Sections - Quick Start Guide

## What Are Advanced Sections?
Advanced Quote Sections allow you to add detailed technical and project information to your quotes:
- **Bill of Materials (BOM)**: List all components and parts
- **Service Level Agreement (SLA)**: Define service commitments and metrics
- **Project Timeline**: Map out project phases and milestones

## How to Add Advanced Sections to a Quote

### Step 1: Create or Edit a Quote
1. Navigate to **Quotes** > **Create New Quote**
2. Or edit an existing quote (that hasn't been invoiced)

### Step 2: Fill Basic Information
1. Complete the required fields:
   - Select client
   - Add line items
   - Set pricing and taxes

### Step 3: Add Advanced Sections
1. Scroll down to find the **"Advanced Sections (Optional)"** card
2. You'll see three tabs: **Bill of Materials**, **SLA**, and **Timeline**

### Step 4: Add Bill of Materials (BOM)
1. Click the **Bill of Materials** tab
2. Click **"Add Item"** button
3. For each component, enter:
   - Part Number (required)
   - Description (required)
   - Manufacturer
   - Quantity and Unit of Measure
   - Technical specifications
   - Notes
4. Add multiple items as needed
5. Remove items using the trash icon

### Step 5: Add Service Level Agreement (SLA)
1. Click the **SLA** tab
2. Fill in the service commitments:
   - Overview of SLA
   - Response Time (e.g., "4 business hours")
   - Resolution Time (e.g., "24 business hours")
   - System Availability (e.g., "99.9% uptime")
   - Support Hours (e.g., "24/7" or "Mon-Fri 9AM-6PM")
   - Escalation Process
3. Add performance metrics:
   - Click **"Add Metric"**
   - Enter metric name, target, and description
   - Select measurement unit
   - Specify penalty for non-compliance

### Step 6: Add Project Timeline
1. Click the **Timeline** tab
2. Enter project overview and dates:
   - Project Overview
   - Project Start Date
   - Project End Date
3. Add milestones:
   - Click **"Add Milestone"**
   - Enter milestone name and description
   - Set start/end dates and duration
   - Select status (Planned, In Progress, Completed, Delayed)
   - List deliverables and dependencies
4. Add multiple milestones for each project phase

### Step 7: Save the Quote
1. Click **"Create Quote"** or **"Update Quote"**
2. Advanced sections are automatically saved

## Viewing Advanced Sections

### On Quote Detail Page
- Navigate to any quote with advanced sections
- Sections appear after Terms & Conditions
- Only non-empty sections are displayed
- All data is shown in a clean, organized format

### In PDF Exports
- Click **"Download PDF"** on any quote
- Advanced sections are automatically included
- BOM, SLA, and Timeline appear on separate pages if needed
- Perfect for client presentations

### In Email
- Click **"Email Quote"**
- The PDF attachment includes all advanced sections
- Recipients see the complete quote with technical details

## Common Use Cases

### Software Development Quote
- **BOM**: Server hardware, licenses, development tools
- **SLA**: Response times, uptime guarantees, bug fix commitments
- **Timeline**: Requirements, design, development, testing, deployment phases

### Hardware Installation Quote
- **BOM**: All equipment, cables, mounting hardware, tools
- **SLA**: Installation warranty, support hours, repair time
- **Timeline**: Site survey, installation, testing, training phases

### Consulting Services Quote
- **BOM**: Not typically needed
- **SLA**: Availability, deliverable turnaround times
- **Timeline**: Discovery, analysis, recommendation, implementation phases

### Custom Manufacturing Quote
- **BOM**: Raw materials, components, fasteners, finishes
- **SLA**: Quality standards, defect handling
- **Timeline**: Design approval, prototyping, production, delivery

## Editing Advanced Sections

### To Modify Existing Sections
1. Click **Edit** button on quote detail page
2. Scroll to Advanced Sections
3. Make your changes
4. Click **Update Quote**

### To Remove Sections
1. Edit the quote
2. Navigate to the section tab
3. Remove all items/metrics/milestones
4. Or clear all text fields
5. Save - empty sections won't be displayed

### To Copy Sections Between Quotes
Currently manual - future enhancement planned:
1. Open the source quote
2. Copy relevant section data
3. Create/edit target quote
4. Paste the information

## Troubleshooting

### Sections Not Saving
- Ensure you click "Create Quote" or "Update Quote"
- Check that required fields in sections are filled
- Verify you're not editing an invoiced quote

### Sections Not Appearing in PDF
- Refresh the page and try downloading again
- Verify sections have content (not all fields empty)
- Check browser console for errors

### Lost Data After Editing
- Always click Save/Update before navigating away
- Use browser back button carefully
- Draft status quotes can be edited freely

## Keyboard Shortcuts

While in Advanced Sections:
- **Tab**: Move between fields
- **Shift+Tab**: Move backwards
- **Enter**: Submit forms (be careful in text areas)
- **Esc**: Close dialogs
